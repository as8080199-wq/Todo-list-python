# 📝 Simple To-Do List App (Python)

This is a beginner-friendly To-Do List application built using Python.
It runs in the command line and allows users to manage daily tasks.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* 🔁 Simple menu-based system

---

## 🛠️ Technologies Used

* Python 3

---

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the program:

```bash id="91zxlh"
python todo.py
```

4. Choose options from the menu
